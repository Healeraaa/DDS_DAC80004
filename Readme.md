# CV通信协议

| 字段   | 类型           | 说明         |
|--------|----------------|--------------|
| 帧头   | 0xFF           | 帧起始标志   |
| 命令   | uint8_t(0x00)  | 协议命令字   |
| 参数1  | double         | 初始电位     |
| 参数2  | double         | 终止电位     |
| 参数3  | double         | 扫描极限1    |
| 参数4  | double         | 扫描极限2    |
| 参数5  | double         | 扫描速率     |
| 参数6  | double         | 循环次数     |
| 参数7  | None           | 保留参数     |
| 参数8  | None           | 保留参数     |
| 参数9  | None           | 保留参数     |
| 参数10 | None           | 保留参数     |
| 帧尾   | 0xFE           | 帧结束标志   |
